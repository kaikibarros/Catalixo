Arquitetura Técnica do Sistema

O projeto foi construído utilizando uma arquitetura de cliente-servidor moderna, garantindo que o processamento
pesado de imagens seja feito no backend, enquanto o usuário interage com uma interface leve e responsiva.

Como a Arquitetura Funciona

1. Frontend

O usuário faz upload de uma imagem do resíduo através da nossa interface web. A imagem é capturada pelo navegador e preparada para envio.

2. FastAPI

A imagem é enviada via requisição HTTP POST para o nosso servidor em Python. O FastAPI gerencia a requisição e repassa os dados para a IA.

3. Classificação Visão Computacional

O modelo analisa as características visuais do objeto e retorna a categoria exata (ex: Papel, Plástico) para a tela do usuário.

A Magia da Visão Computacional

A Visão Computacional é o cérebro do nosso sistema. É essa tecnologia que permite à aplicação olhar para um objeto por uma imagem e entender do que ele é feito.

Utilizamos um modelo pronto, com o apoio do Hugging Face para interfência. A classificação do lixo refere-se à separação de vários tipos de resíduos em diferentes categorias, de acordo com o impacto ambiental do seu uso, o valor da composição dos componentes do lixo e os métodos de tratamento existentes.

O conjunto de dados de classificação de lixo é do Kaggle. Há um total de 2467 imagens neste conjunto de dados. E este modelo é um modelo de classificação de imagens para este conjunto de dados. Existem 6 classes para este conjunto de dados, que são papelão, vidro, metal, papel, plástico e lixo.

  • 🎯 Precisão: Capacidade de distinguir entre um copo de plástico reciclável e um não reciclável.
  • 🌍 Impacto: Reduz o erro humano na hora da separação, garantindo que o lixo vá para o destino sustentável correto.
